shiro结合ehcache实现缓存

Description

为解决每次调用接口都执行shiro授权操作，影响效率，因此引入缓存概念

STEP

  1. 导入依赖
  2. 配置ehcache-shieo.xml文件
  3. ShiroConfig类中添加相关Bean
    1. EhCacheManager
    2. DefaultWebSecurityManager（设置缓存管理器，即a）
    3. UserRealm中启用验证，名字为xml文件中的
  4. 启动类开启缓存
